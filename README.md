# Fullstack Content Platform

A modern MERN stack content management platform where users can create, upload, view, and delete posts with images.

---

# Features

- Create posts with image upload
- Delete posts
- Dynamic post rendering
- Responsive modern UI
- REST API integration
- MongoDB database connection
- Image upload support using Multer
- Fully responsive design

---

# Tech Stack

## Frontend
- React
- Tailwind CSS
- Lucide React Icons

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
